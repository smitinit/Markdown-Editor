Markdown Editor
Press Ctrl + Y to open toolbar.
Press Ctrl + M to open documents.

Instructions:

Navigate using Arrow Keys: ↑ Up Arrow / ↓ Down Arrow

Backspace: Removes a word and moves to the previous line if the current line is empty.

Ctrl + Enter: Inserts a new line below the cursor.

Ctrl + B: Deletes the current line.

Automatically adds new lines at the bottom.

Supports Add and Delete buttons.
